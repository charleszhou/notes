nvm和n

nvm设置与使用

安装

nvm

https://github.com/creationix/nvm

nvm-windows

https://github.com/coreybutler/nvm-windows

设置镜像加速

linux

export NVM_NODEJS_ORG_MIRROR=https://npm.taobao.org/mirrors/node

source ~/.nvm/nvm.sh    //添加到.bashrc或.bash_profile或.profile中

windows

使用命令nvm root找到nvm的安装目录

目录下有一个setting.txt，在后面加上这两行

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

使用 .nvmrc 文件配置项目所使用的 node 版本

cd <项目根目录>    #进入项目根目录
echo 4 > .nvmrc   #添加 .nvmrc 文件（数字代表版本号）
nvm use           #无需指定版本号，会自动使用 .nvmrc 文件中配置的版本
node -v           #查看 node 是否切换为对应版本

npm镜像加速设置

安装好nodejs后，需要用npm安装组件。但是由于众所周知的原因，速度非常慢，需要将npm的镜像加速。只要将默认镜像换成淘宝的镜像就可以了。
在用户主目录下创建“.npmrc”文件（默认是没有这个文件的），然后加入
加入下面内容：
registry = https://registry.npm.taobao.org
或者使用命令（ubuntu16.04.2已经测试成功）
npm config set registry https://registry.npm.taobao.org

原文链接:
加速npm镜像
http://www.cnblogs.com/Irving/p/4083676.html

nrm管理镜像

全局安装

npm i nrm -g

让你的nodejs代码修改后无需重启应用即可生效的工具supervisor

node-sass安装失败后的解决办法

npm uninstall node-sass && npm i node-sass --sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

ubuntu下mongodb的安装

安装方法：

Robo 3T跨平台开源 MongoDB 可视化管理工具安装（ubtuntu）

和16.04有冲突暂时不安装

下载地址：https://robomongo.org/

安装命令：sudo tar -C /usr/local –strip-components 1 -xzf robo3t-1.1.1-linux-x86_64-c93c6b0.tar.gz

mongodb默认只能本机访问，通过修改/etc/mongodb.conf文件，让所有ip都可以访问
注释掉bindip:127.0.0.1

如果要使用gui工具管理mongodb，可以在windows上运行robo 3T
