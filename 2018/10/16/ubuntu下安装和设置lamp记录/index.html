ubuntu下安装和设置lamp记录

发表于 2018-10-16 | 更新于 2018-11-21 | 分类于 运维 ， 操作系统

安装

先安装mysql

sudo apt-get install mysql-server

在安装的最后会提示输入mysql的root密码

受益于apt-get会自动解决软件包依赖，安装lamp只需要一个命令

sudo apt-get install phpmyadmin

就可以安装完lamp的其它组件

安装时会提示新建一个数据库，可以选是也可以不安装

如果phpmyadmin安装完后，并不在apache默认路径下，需要建立一个连接

sudo ln -s /usr/share/phpmyadmin /var/www/html

创建、启动、禁用和删除站点

站点配置文件在/etc/apache2/site-available中，每个站点一个配置文件，创建文件就是创建站点，删除文件就是删除站点。
启动和停用站点的命令

这两个命令实际就是在/etc/apache2/site-enabled目录中创建和删除相应站点配置文件的软链接。

设置好站点后需要重启apache才能使配置生效。

调整目录权限

根据教程：安装 LAMP Web 服务器（在 Amazon Linux 上）介绍的方式来处理web目录的权限。通过新建一个群组www，然后赋予该组目录的所有权并添加写入权限。随后，该组的成员都将能够为 Web 服务器添加、删除和修改文件了。

目录设置的目标

  1. 可以上传网站(如果是一个人管理使用服务器就使用scp上传下载文件，多人使用可以配置pureftp)
  2. web可以访问可以通过网站上传文件
  3. 管理员可以修改文件

操作步骤：
1、新建一个群组

sudo groupadd www

2、将用户admini添加到www群组（新建群组后需要注销用户权限才能生效）

sudo usermod -a -G www admini

3、将apache的用户添加到www群组

sudo usermod -a -G www www-data

4、将web目录的所有权更改到www群组

sudo chown -R root:www /var/www

5、更改 /var/www 及其子目录的目录权限，给群组用户添加写权限

sudo chmod -R g+w /var/www
